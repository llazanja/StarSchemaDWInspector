# StarSchemaDWInspector

SETUP

# Clone the repository
```
    git clone git@github.com:llazanja/StarSchemaDWInspector.git
```

# Position yourself inside the project root directory
```
    cd StarSchemaDWInspector/
```

# Install dependencies
```
    npm install
```

# Run server
```
    npm run sstart
```

# Run client
```
    npm run cstart
```
